How do you install The Lab Kodi Addon?

Install the Lab Repository

  1. You may have to¬†make sure that you enable sources from unidentified sources¬†to be available in the Kodi settings.
  2. Enter Your Kodi settings and then open the file manager.
  3. Choose “Add Source” and then click “None.”
  4. Type in the path “https://th3l48.github.io/TheLab/" and select “Done”
  5. Click on the box to highlight it and then type in an appropriate name, such as “TheLab,” for this media source.
  6. Click “OK.”
  7. Return to the main menu, and then select “Addons” on the left-hand side.
  8. Start the browser for add-ons by clicking on the small box symbol on the upper left
  9. Choose “Install from the ZIP file.”
  10. Respond to the warning message with “Yes.”
  11. Open the source “TheLab” and select the file “repository.thelab-x.x.x.zip.”
  12. Then, wait until a message appears that the repository was installed successfully.

If you’re not sure what to do to install the Kodi add-on,¬†look up our tutorial including pictures.

Setup The Lab Kodi Addon

  1. In your browser, for add-ons, click “Install using repository.”
  2. Choose the one that is installed, “The Lab Repository.”
  3. Navigate to the category “Video add-ons”
  4. Select the Lab and then click “Install” on the right side.
  5. Verify the installation of dependencies by choosing “OK.”
  6. It would help if you waited until you saw a notification that indicates the Kodi addon has been installed successfully.
  7. Go back to the menu, and the Kodi addon will be available via the main menu.
