How do I install the Kodzi Kodi Addon?

Install The Kodz Repository

  1. You might need to¬†allow the unknown source¬†within your Kodi settings.
  2. You can go into your Kodi settings and then open the file manager.
  3. Choose “Add source” and click on “None.”
  4. Type in the path “https://mrkodz.github.io/repo/” and select “Done”
  5. Click on the box to highlight it and then type in initials like “MrKodz” for this media source
  6. Click “OK.”
  7. Return to the main menu and select “Addons” on the left.
  8. You can open the addon browser by clicking the little box icon at the top left
  9. Choose “Install from the ZIP file.”
  10. Open the source “MrKodz” and select the file repository.thekodz.zip
  11. You should wait until you see a notification that the repository was installed successfully.

If you’re unsure how to install the Kodi addon,¬†look up our guide with screenshots.

Install Kodzi Kodi Addon

  1. In the browser for your addon, click “Install from repository.”
  2. Choose the available one, “The Kodz Repository.”
  3. Select the section “Program addons.”
  4. Choose Kodzi and then click “Install” on the bottom right
  5. KODI 18 ONLY Make sure you have confirmed the installation of dependencies by choosing “OK.”
  6. You should wait until you see a notification that indicates the Kodi addon was successfully installed.
  7. Return to the main menu, and the Kodi addon will be available.

Install a zip archive using Kodzi

  1. Begin your Kodzi Kodi extension and click “Enter Website URL.”
  2. Copy or paste the link to the zip file.
  3. Be sure that the link ends with “.zip” and points to the Kodi extension or repository
  4. Kodzi installs this zip file on your behalf, and then you can use the addon or repository like you normally
