How to install the Crew Kodi Addon

Install The Crew Repository

  1. You might need to¬†allow unidentified sources¬†within your Kodi settings.
  2. You can go into Your Kodi settings and then open the file manager.
  3. Choose (Add source) & click on “None.”
  4. in the path, type “https://team-crew.github.io/" and select “Done”
  5. Click on the box to highlight it and enter an appropriate name, such as “the_crew” for this media source
  6. Click “OK”
  7. Return to the main menu and click “Addons” on the left.
  8. You can open the addon browser by clicking the small box symbol on the upper left
  9. Choose “Install from a ZIP file.”
  10. Refuse to ignore the warning message “Yes.”
  11. Open on source “the_crew” and select the file “repository.thecrew-x.x.zip”
  12. It would help if you waited until you saw a notification that the repository was installed.

Install The Crew Kodi Addon

  1. In the addon browser, select “Install from repository.”
  2. Choose the one that is the option to install “The Crew Repo.”
  3. Select the tab “Video addons.”
  4. Choose The Crew and click on “Install” on the bottom right
  5. Verify the installation of dependencies by clicking “OK.”
  6. It would help if you waited until you saw a notification that indicates the Kodi addon has been installed successfully.
  7. Return to the main menu, and the Kodi addon will be available.

How can I solve the problem of the Crew not working with Kodi for Windows?

It is possible that the Crew Kodi Addon does not work on Kodi on Windows and closes by displaying an error.¬†For other platforms, the addon functions without issues.

The reason is that Windows’ integrated security against viruses does not correctly identify a related file as malicious and remove it.

When you open Windows Security in the settings and click “Virus & threat protection,” you can see that a file has been classified as “Trojan: Python/MCCrash.B!MTB.”

Use these suggestions to fix the issue:

  • On Windows, look up “Windows Security” and click on the result at the top.
  • Select “Virus & threat protection.”
  • In"Current Threats,” select the “Current threats” section, and choose “Protection history.”
  • Look for a blocked file detected as “Trojan: Python/MCCrash.B!MTB.”
  • Select “Actions” and unblock the file.

Important: Before unblocking any file, ensure the timestamp you use is correct.¬†Don’t block files that aren’t compatible with Kodi or specific addons.¬†If you’re in doubt, don’t take any action in any way.
