What are Kodi extensions?

Addons can enhance the Kodi experience dramatically by adding features for your media center.¬†Certain addons can use to broadcast audio and video content on the internet or to maintain the integrity of your Kodi installation by eliminating any cache files and temporary ones.

There are two kinds of Kodi addons: official and unofficial.¬†Official Kodi addons can be downloaded through the Kodi Addon Repository, which comes installed by default.¬†The Kodi website has aninventory of all officially-approved addons.

Unofficial Kodi addons need to be installed by third-party repositories. This is done via the inclusion of sources into the file management program.

Install Kodi addons in three easy steps

The majority of Kodi addons are installed with just three simple steps:

  • In addition to a source
  • Setting up the repository
  • Installing the extension from the repository

Before we tell you what you need to do in order to add Kodi addons, make sure you allow unknown sources to be enabled in the settings.¬†With our¬†guide, we will walk you through step-by-step¬†steps to accomplish this.

How do you include a source into Kodi, the file manager

  1. Launch Kodi and click an icon for gear to access the options.
  2. Start your file manager by clicking the folder icon.
  3. Select “Add Source” (may require a double-click on certain devices).
  4. If you are able to see this window, select “. "
  5.   7. Click “OK” to confirm your input. Step 1 is complete, and you’ve successfully added an external source.

How do I install a Kodi repository?

  1. On the Kodi home screen, select “Addons” on the left side.
  2. Click on the box icon at the top left corner to launch"Addon Browser.”
  3. Click “Install from zip file.”
  4. The Kodi file manager is opened and displays the list of connected devices as well as your sources.¬†Select the source we added in the previous step.
  5. Now, you have to locate the zip file that includes the word “repository” and the name of the repository that you need.Sometimes the zip file is not able to be found on the initial layer, but instead in subfolders inside the source.

The file names for repositories are always in this format: “repository.repositoryname-x.x.x.zip.”

Step 2 is completed, and you’ve successfully added an external Kodi repository.

How do I install Kodi addons?

  1. Upon successful installation of the repository, select “Install from repository” in the “Addon Browser.”
  2. A listing of the installed Kodi repositories will appear.¬†Select the one you’ve recently installed.
  3. You now need to choose the category of the Kodi addon you’d like to install.¬†Most of the time, the addon is listed within the “Video addons” category.¬†In this guide, however, the addon is located in “Program addons add-ons.”
  4. Choose the Kodi addon you would like to install.¬†The majority of Kodi repositories have a lot of add-ons, so ensure you select the correct one.
  5. Click “Install” at the bottom right.
  6. If running Kodi 18 Leia and later, it will display a window that shows you the dependencies that will be installed, along with the add-on.¬†These dependencies need to be installed or else the Kodi addon will not be functioning properly.¬†Click “OK” on the left side.
  7. After a couple of seconds, the Kodi addon has been installed successfully.
  8. When you come back to the home screen of Kodi, every addon installed will be visible once you tap"Addons," which is the option in the menu “Addons” on the left side.
